# Telegram Bot для продажи файлов оптимизации ПК

Рабочий проект Telegram-бота на Python + aiogram 3. Бот автоматически регистрирует пользователей по Telegram ID, показывает личный кабинет, каталог с вложенной структурой, демо-файлы после проверки подписки, покупки через Telegram Stars и CryptoBot, подписки, FAQ, поддержку, отзывы и админ-панель.

## Возможности

- Авторизация по Telegram ID и хранение даты регистрации, баланса и количества покупок.
- Личный кабинет: профиль, история покупок, активные подписки.
- Каталог по схеме: раздел → подкатегория/игра → тип оптимизации → товар.
- Раздел FAQ и визуальные метки товаров: `🔥 Популярное`, `🆕 Новинка`, `💎 Рекомендуем`.
- Демо-версии с обязательной проверкой подписки на канал.
- Защита от повторной бесплатной выдачи демо и повторного начисления одной покупки.
- Покупка через Telegram Stars и CryptoBot.
- Экстро-оптимизация с обязательным предупреждением и кнопкой `⚠️ Я понимаю риски`.
- Выдача restore-файла или инструкции для отката экстро-настроек.
- Админ-панель: добавление товара через FSM, просмотр товаров, пользователей, продаж и подписок.

## Быстрый запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Заполните `.env`:

- `BOT_TOKEN` — токен от `@BotFather`.
- `ADMIN_IDS` — Telegram ID администраторов через запятую.
- `REQUIRED_CHANNEL` — канал для проверки подписки перед выдачей демо.
- `SUPPORT_USERNAME` — username администратора.
- `REVIEWS_CHANNEL_URL` — ссылка на канал отзывов.
- `CRYPTOBOT_TOKEN` — токен CryptoBot, если нужна оплата криптой.

Запуск:

```bash
python main.py
```

## Файлы товаров

Стартовый каталог создаётся автоматически при первом запуске. Пути к файлам указаны в `bot/services/catalog_seed.py`.

Разложите реальные файлы и скриншоты по папкам:

- `data/assets/full/` — полные версии.
- `data/assets/demo/` — демо-версии.
- `data/assets/photos/` — фото товаров и реальные скриншоты результата.

Если файл ещё не загружен, бот не падает: он отправит пользователю уведомление, какой путь не найден.

## Telegram Stars

Для Stars используется валюта `XTR`. Бот проверяет `pre_checkout_query`, а полный файл выдаётся только после `successful_payment`.

## CryptoBot

CryptoBot создаёт invoice через Crypto Pay API. После оплаты пользователь нажимает `Проверить оплату`, бот запрашивает статус invoice и только при `paid` выдаёт полный файл.

## Админ-панель

Админ-панель доступна пользователям из `ADMIN_IDS`. Через кнопку `➕ Добавить товар` бот последовательно запросит:

- название;
- описание;
- фото и скриншот;
- полный файл и демо-файл;
- цену;
- категорию, подкатегорию и тип оптимизации;
- FPS до/после;
- метку;
- признак экстро-оптимизации;
- restore-файл или инструкцию.

## Важное

Для проверки подписки бот должен быть администратором канала или иметь доступ к списку участников. Для реального магазина замените seed-файлы, фото и скриншоты на настоящие материалы.
